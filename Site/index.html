Characters that you should always escape

  • <
    • Use &_lt; with the _ removed to create the symbol
      • If typed with an uppercase L then a double less than sign is created
  • >
    • Use "&_gt ;" with the _ removed to create the symbol
      • If typed with an uppercase G than a double greater than symbol is created
  • &
    • Use "&_amp ; with the _ removed to create the symbol
  • ©
      Use "&_copy ;" with the _ removed to create the symbol
  • Non wrapping Space i.e the spaces between words are removed so that when wrapping occurs the words move together to the next line
    • Example: Word1&nbsp;Word2&nbsp;Word3

Creating Links

Internal Links

We can link to a file in the same directory as this HMLT file like this: Linking to a file in the same directory
  • The text that follows the < sign that ends the opening statement is what will appear as the text for the link
  • A basic vizualization of the structure <a> href="file path" title="A title"> Text that will appear as the link</a>
    • You can also use DIV within <a and /a> to place the link on the line below
      • Example:
        Linking a file in the same directory
    • If a direct file path or webpage is not given HTML will assume that the file that is placed under "file path" will be in the same directory

External Links

We can link to a different website by using href="Web Address" target="" title="" within the <
  • When target is set to "_blank" a new tab is created when the link is clicked

Fragment Identifier

When creating a link within a webpage that references another point in the same webpage you can assign an id to any tag in the html that will be referenced by the link
Example:
  • FragmentID
    • When creating the id attached to the tag the # symbol is not used but it is used in the href= portion of the html
